# wstoggler

## Install
wstoggler requires `golang 1.16`

1. `git clone git@github.com:deifyed/wstoggler.git && cd wstoggler`
2. `make install`
    1. Use INSTALL_DIR to customize installation directory. Default is `~/.local/bin`
   
## Usage
### Moving to workspace
```shell
# Syntax
wstoggler <workspace>

# Example
wstoggler 1
```

### Move to previous workspace
```shell
# Syntax
wstoggler

# Example
wstoggler
```
